# SplitPay — Brand Kit (v2, 100% vectorial)

Todos los archivos fueron redibujados como vectores limpios: bordes nítidos
a cualquier tamaño, sin artefactos.

## Archivos
| Archivo | Uso |
|---|---|
| logo.svg / logo.png | Logo principal (mascota + wordmark + tagline), fondo transparente |
| logo-light.png | Para fondos claros (texto navy) |
| logo-dark.png | Para fondos oscuros (texto blanco) |
| mascot.svg / mascot.png | Mascota sola |
| mascot-fun.svg / mascot-fun.png | Mascota sentada con lentes y refresco |
| favicon.svg / favicon.ico | Favicon (ico: 16/32/48/64 px) |
| app-icon-1024/512/192.png | Íconos de app (círculo verde, esquinas transparentes) |
| apple-touch-icon.png | 180×180, fondo blanco (iOS no admite transparencia) |
| email-logo.png | Lockup horizontal para correos (1200 px) |
